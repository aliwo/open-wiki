**로그인**
----
    
    Oauth 로그인이 아닌 일반 계정의 로그인

* **URL**

  /user/login

* **Method:**
  
  `POST`
  
*  **URL Params**
    
    N/A

   **Required:**
 
    N/A

   **Optional:**
 
    N/A

* **Data Params**

  ```json
    {
        "account": "계정 아이디",
        "password": "비밀번호"
    }
   ```

* **Success Response:**
  
  * **Code:** 200 <br />
    **Content:** `{ "id" : 12 }`
 
* **Error Response:**

  * **Code:** 400 Bad Request <br />
    **Content:** `{ "error" : "Wrong Passwrod" }`

* **Sample Call:**

  ```json
    {
        "account": "recordable",
        "password": "my_password"
    }
   ```
