* **URL**

  /clip/heart/ranking/<int:clip_id>

* **Method:**
  
  | `GET` |
  
*  **URL Params**

   **Required:**
 
   N/A
   
   **Optional:**
 
   N/A

* **Data Params**

    N/A
    
    **Optional:**
    N/A

* **Success Response:**
  
  * **Code:** 200 <br />
    **Content:** 
    ```json
    {

      "okay":true, 
      "ranking": [
        {"user": "<유저 객체>", "heart": 1500},
        {"user": "<유저 객체>", "heart": 1000}
      ]
    }
    ```
 
* **Error Response:**

  * **Code:** HTTP_404_NOT_FOUND <br />
    **Content:** 
    ```json
        {
            "ok": false,
            "msg": "object not found"
        }
    ```
    해당 id의 clip 이 존재하지 않는 경우
